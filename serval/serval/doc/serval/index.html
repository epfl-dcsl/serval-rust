Serval

Serval [1] is a framework written in Rosette for creating automated verifiers for low-level systems code. Serval comes with verifiers for the RISC-V, x86-32, LLVM, BPF instruction sets. Serval provides a library for writing system specifications, such as state-machine refinement and noninterference.

To develop your own tools, you may want to familiar yourself with the Rosette language. The Rosette Guide is a good starting point.
