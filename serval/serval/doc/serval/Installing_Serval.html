1 Installing Serval

The easiest way to play with Serval is using a Docker image. Follow the instructions on the SOSP’19 artifact page.

You may also be interested in the SOSP’19 tutorial on how to verify a toy security monitor using Serval.

Alternatively, you can install (actively developed version) of Serval from source using Racket’s raco tool:

  • Download and install Racket (version 7.5 or later).

  • Git clone the source code in the “serval” directory.

  • Uninstall any previous versions of Serval:

    > raco pkg remove serval

  • Install Serval:

    > cd serval

    > raco pkg install
